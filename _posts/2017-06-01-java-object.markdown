## 什么是面向对象？

- 1967年 Simula 提出面向对象概念。

- 编程是为了解决问题而存在的。

- 如果把一个问题看成对象，则可以拆分为基本两种。

  问题需要解决的条件，即存在的属性或特征。

  问题需要解决的动作或功能，即需要用什么方法来解决这个问题。
