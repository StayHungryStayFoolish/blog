## 计算机语言发展

- 1GL

    二进制机器语言 0/1 （机器来阅读0 、 1 从而进行计算, 编程晦涩难懂）

- 2GL LLL Low Level Language

    低级计算机语言

    eg. 汇编语言 （可以转换为机器可读的语言, 更方便人来读写代码）

- 3GL HLL High Level Language

    高级计算机语言 （更容易被人所理解）

    eg. Java C C++ Python Go

- 4GL

    eg. SQL 解释型语言（通俗易懂）
